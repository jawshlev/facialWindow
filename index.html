Refor(me)d

Description

This project uses ml5js and p5js to identify facial features of the user and create graphical windows around them.

The graphical windows resize to keep your facial features within them even if you move your face around, and you can click and drag to reform your face however you like.

Controls

  • Click+Drag: Move graphical windows
  • 1 key: Toggle sidebar
  • 2 key: after image mode while dragging
  • t key: Toggle coordinate labels
  • Double-click: Remove a graphic
