Interactive Experience

Description

This is an interactive p5.js experience. The animation shows colorful circles that move in a wave pattern. The experience is designed to be both visually appealing and interactive.

You can interact with the canvas by moving your mouse over it and clicking to create new elements. Use the controls at the top to pause/play the animation or reset it.

Controls

  • Mouse Movement: Move your cursor over the canvas to interact with elements
  • Click: Create a new colored circle at the cursor position
  • Play/Pause Button: Toggle the animation
  • Reset Button: Restart the experience

About

This interactive experience was created using p5.js, a JavaScript library that makes coding accessible for artists, designers, educators, and beginners. It's perfect for creating visual and interactive experiences on the web.

p5.js is inspired by Processing and brings the original goals of Processing to the web. Learn more at p5js.org.
